# ☁️ Air Quality Dashboard: Stasiun Aotizhongxin

Dashboard ini dibuat untuk memvisualisasikan hasil analisis data kualitas udara dari stasiun Aotizhongxin pada periode 2013 hingga 2017. Proyek ini merupakan bagian dari submission kelas Belajar Analisis Data dengan Python.

## Setup Environment

Sebelum menjalankan dashboard, pastikan Anda telah menginstal semua library yang dibutuhkan. Buka terminal atau command prompt, arahkan ke folder proyek ini, lalu jalankan perintah berikut:
pip install pandas matplotlib seaborn streamlit

## Cara Menjalankan Dashboard
Setelah semua library berhasil diinstal, jalankan perintah berikut di terminal/command prompt Anda:
streamlit run dashboard.py

Dashboard akan secara otomatis terbuka di browser default Anda (biasanya di http://localhost:8501).
